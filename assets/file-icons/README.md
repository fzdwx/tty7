These SVGs are vendored from `pierrecomputer/vscode-icons`, the VS Code file
icon theme used by the Pierre ecosystem around `diffs.com`.

- Source: https://github.com/pierrecomputer/vscode-icons
- Commit: `04a9028f0b227aaf820e9e73da2992af86ba0f26`
- Build: `npm run build`, using the generated Complete tier in `icons/`
- License: MIT, see `LICENSE.md`

The Complete tier uses per-icon colors from the Pierre palette. `tty7` embeds
these assets and uses `theme-complete.json` for file name and extension mapping
so the file tree stays aligned with the upstream theme.
